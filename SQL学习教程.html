0x01. SQL简介与安装

1.概述

SQL(Structured Query Language) 结构化查询语言 ， 用于访问和处理数据库的标准计算机语言。

特点:

  • 统一性，不同的数据库稍有不同
  • 非过程化语言
  • 语言简洁
  • 可以与其他语言结合

语法特点:

  • 对关键字的大小写不敏感

  • 可以单行或多行书写 以;结束

  • 注释

数据库系统

  • 关系型数据库(RDBMS): MYSQL等
  • 非关系型数据库(NoSQL): Redis等

0x02. MYSQL基本操作-DDL

1.DDL解释

DDL(data definition Language)数据定义语言,包括:

  • 对数据库的常用操作
  • 对表结构的常用操作
  • 修改表结构

2.对数据库的常用操作

功能 SQL
查看所有的数据库 show databases ;
创建数据库 create database [if not exists] mydb1 [charset=utf8] ; #[]里的内容代表可以省略
切换 数据库 use mydb1 ;
删除数据库 drop database [if exists] mydb1 ;
修改数据库编码 alter database mydb1 character set utf8 ;

0x04 MYSQL 约束

1.基本介绍

约束: constraint

即表中数据的限制条件

作用: 表再设计的时候加入约束的目的就是为了保证表中记录完整性和有效性，比如用户表中有些值不能为空，有些列的值（id）不能重复.

  • 主键约束
  • 自增长约束
  • 非空约束
  • 唯一性约束
  • 默认约束
  • 零填充约束
  • 外键约束
